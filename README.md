# DayZ ReTexturing config
 Very simple config files for retexturing vanilla clothes/gear
 
 Remember to save the config files as "config.cpp" in the same directory as your "texture.paa" file.

 Config files for:
                                Bandana,
                                Beanie,
                                Combat Boots,
                                Combat Knife,
                                Gorka Jacket,
                                Gorka Pants,
                                Tactical Gloves,
                                Tortilla backpack
